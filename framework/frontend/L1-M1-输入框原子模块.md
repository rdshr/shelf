# 前端输入框原子模块:FrontendInputFieldAtoms

@framework

本模块定义输入框这一层的组件原子。它把更底层的槽位壳体、值通道、宿主绑定与令牌主题收束成稳定输入框原子。

## 1. 能力声明（Capability Statement）

- `C1` 输入框外形承载能力：沉淀输入框的文本槽位、边框轮廓与前后缀承载结构。
- `C2` 值状态承载能力：沉淀输入值、占位内容、清空规则与错误状态结构。
- `C3` 绑定承载能力：沉淀输入框与前端框架值绑定、事件回调和可访问标记之间的稳定结构。
- `C4` 非能力项：不负责领域字段语义、校验逻辑实现与具体主题值。

## 2. 边界定义（Boundary / 参数）

- `FIELDCONTENT` 内容边界：标签槽位、占位内容、前缀/后缀与帮助信息位置需稳定。来源：`C1 + C2`。
- `FIELDFRAME` 外形边界：输入框高度、内边距、边框粗细角色、圆角形态与前后缀槽位需稳定。来源：`C1 + C3`。
- `FIELDVALUE` 值边界：输入值格式、默认值、清空规则与只读规则需稳定。来源：`C2 + C3`。
- `FIELDSTATE` 状态边界：默认、聚焦、禁用、只读与错误状态需稳定。来源：`C1 + C2`。
- `FIELDBIND` 绑定边界：宿主元素、value/defaultValue、onChange/onFocus/onBlur/ref 等接口需稳定。来源：`C2 + C3`。
- `FIELDA11Y` 可访问边界：label 关联、描述信息、焦点顺序与错误提示需达标。来源：`C1 + C3`。

## 3. 最小可行基（Minimum Viable Bases）

- `B1` 输入框外壳结构基：L0.M0[R2,R3] + L0.M3[R1,R2]。来源：`C1 + FIELDCONTENT + FIELDFRAME`。
- `B2` 输入值状态基：L0.M1[R1,R3]。来源：`C1 + C2 + FIELDVALUE + FIELDSTATE`。
- `B3` 输入绑定结构基：L0.M2[R1,R3]。来源：`C2 + C3 + FIELDBIND + FIELDA11Y`。

## 4. 基组合原则（Base Combination Principles）

- `R1` 外壳先行
  - `R1.1` 参与基：`B1`。
  - `R1.2` 组合方式：先固定输入框文本槽位、边框轮廓和前后缀承载结构，再允许更上层结构附着字段语义。
  - `R1.3` 输出能力：`C1`。
  - `R1.4` 边界绑定：`FIELDCONTENT/FIELDFRAME/FIELDA11Y`。
- `R2` 值状态闭合
  - `R2.1` 参与基：`B1 + B2`。
  - `R2.2` 组合方式：在输入外壳上绑定默认值、占位规则、清空规则与状态集合，形成稳定输入值结构。
  - `R2.3` 输出能力：`C1 + C2`。
  - `R2.4` 边界绑定：`FIELDVALUE/FIELDSTATE/FIELDFRAME`。
- `R3` 框架绑定闭合
  - `R3.1` 参与基：`B1 + B2 + B3`。
  - `R3.2` 组合方式：将输入外壳和值状态映射到值绑定接口、事件回调与可访问标记，形成可被 React 等前端框架直接承接的输入框原子。
  - `R3.3` 输出能力：`C1 + C2 + C3`。
  - `R3.4` 边界绑定：`FIELDBIND/FIELDA11Y/FIELDVALUE/FIELDSTATE`。
- `R4` 禁止组合
  - `R4.1` 参与基：`B1 + B2 + B3`。
  - `R4.2` 组合方式：跳过输入外壳直接定义值绑定、状态集合不可解释或宿主元素与值接口错配的方案均无效。
  - `R4.3` 输出能力：`C4`。
  - `R4.4` 边界绑定：`FIELDCONTENT/FIELDFRAME/FIELDVALUE/FIELDSTATE/FIELDBIND/FIELDA11Y`。

## 5. 验证（Verification）

- `V1` 外壳完整性：文本槽位、前后缀槽位与边框轮廓必须完整。
- `V2` 值状态完整性：默认值、占位内容、清空规则与状态集合必须满足 `FIELDVALUE/FIELDSTATE` 约束。
- `V3` 绑定有效性：值绑定接口、事件回调与宿主元素必须满足 `FIELDBIND` 约束并能映射到具体前端框架。
- `V4` 可访问符合性：有效组合必须满足 `FIELDA11Y` 约束。
- `V5` 最小必要性：移除任一 `B*` 后，`V2~V4` 至少一项失败。
- `V6` 结论表达：逐条输出 `R* -> C* / 边界` 的通过或失败结论。
