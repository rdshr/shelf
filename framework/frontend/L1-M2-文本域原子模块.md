# 前端文本域原子模块:FrontendTextareaAtoms

@framework

本模块定义文本域这一层的组件原子。它在更底层的槽位壳体、状态通道、宿主绑定与令牌主题之上，增加多行滚动和提交相位约束。

## 1. 能力声明（Capability Statement）

- `C1` 文本域外形承载能力：沉淀多行输入区域、内边距、边框轮廓与计数信息承载结构。
- `C2` 多行状态承载能力：沉淀滚动、扩展高度、占位内容与多行提交状态结构。
- `C3` 绑定承载能力：沉淀文本域与前端框架值绑定、输入回调和可访问标记之间的稳定结构。
- `C4` 非能力项：不负责领域文本语义、富文本编辑器能力与具体主题值。

## 2. 边界定义（Boundary / 参数）

- `AREACONTENT` 内容边界：标签槽位、占位内容、帮助信息与计数位需稳定。来源：`C1 + C2`。
- `AREAFRAME` 外形边界：多行高度、最小高度、边框粗细角色、圆角形态与内边距需稳定。来源：`C1 + C3`。
- `AREASCROLL` 滚动边界：滚动区、自动扩展策略与滚动条策略需稳定。来源：`C1 + C2`。
- `AREASTATE` 状态边界：默认、聚焦、禁用、错误与提交中状态需稳定。来源：`C2 + C3`。
- `AREABIND` 绑定边界：宿主元素、value/defaultValue、onChange/onFocus/onBlur/ref 与提交出口接口需稳定。来源：`C2 + C3`。
- `AREAA11Y` 可访问边界：label 关联、描述信息、键盘换行策略与错误提示需达标。来源：`C1 + C3`。

## 3. 最小可行基（Minimum Viable Bases）

- `B1` 文本域外壳结构基：L0.M0[R2,R3] + L0.M3[R1,R2]。来源：`C1 + AREACONTENT + AREAFRAME`。
- `B2` 文本域滚动状态基：L0.M1[R2,R3]。来源：`C1 + C2 + AREASCROLL + AREASTATE`。
- `B3` 文本域绑定结构基：L0.M2[R1,R3]。来源：`C2 + C3 + AREABIND + AREAA11Y`。

## 4. 基组合原则（Base Combination Principles）

- `R1` 外壳先行
  - `R1.1` 参与基：`B1`。
  - `R1.2` 组合方式：先固定文本域外壳、边框轮廓和计数承载位，再允许更上层结构附着文本语义。
  - `R1.3` 输出能力：`C1`。
  - `R1.4` 边界绑定：`AREACONTENT/AREAFRAME/AREAA11Y`。
- `R2` 滚动状态闭合
  - `R2.1` 参与基：`B1 + B2`。
  - `R2.2` 组合方式：在文本域外壳上绑定滚动区、自动扩展策略与状态集合，形成稳定多行输入结构。
  - `R2.3` 输出能力：`C1 + C2`。
  - `R2.4` 边界绑定：`AREASCROLL/AREASTATE/AREAFRAME`。
- `R3` 框架绑定闭合
  - `R3.1` 参与基：`B1 + B2 + B3`。
  - `R3.2` 组合方式：将文本域外壳和值状态映射到绑定接口、提交出口与可访问标记，形成可被 React 等前端框架直接承接的文本域原子。
  - `R3.3` 输出能力：`C1 + C2 + C3`。
  - `R3.4` 边界绑定：`AREABIND/AREAA11Y/AREASCROLL/AREASTATE`。
- `R4` 禁止组合
  - `R4.1` 参与基：`B1 + B2 + B3`。
  - `R4.2` 组合方式：跳过文本域外壳直接绑定滚动逻辑、滚动策略不可解释或宿主元素与多行绑定接口错配的方案均无效。
  - `R4.3` 输出能力：`C4`。
  - `R4.4` 边界绑定：`AREACONTENT/AREAFRAME/AREASCROLL/AREASTATE/AREABIND/AREAA11Y`。

## 5. 验证（Verification）

- `V1` 外壳完整性：多行内容槽位、边框轮廓与计数承载位必须完整。
- `V2` 滚动状态完整性：滚动区、自动扩展策略与状态集合必须满足 `AREASCROLL/AREASTATE` 约束。
- `V3` 绑定有效性：宿主元素、值绑定接口与提交出口必须满足 `AREABIND` 约束并能映射到具体前端框架。
- `V4` 可访问符合性：有效组合必须满足 `AREAA11Y` 约束。
- `V5` 最小必要性：移除任一 `B*` 后，`V2~V4` 至少一项失败。
- `V6` 结论表达：逐条输出 `R* -> C* / 边界` 的通过或失败结论。
