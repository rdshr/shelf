# 前端视觉语义原子模块:FrontendVisualSemanticAtoms

@framework

本模块定义前端最底层的视觉语义结构。它只负责令牌、主题、密度和可读对比等视觉基础，不再把启动、交付和治理这类运行对象混进同一层。它关注的不是某个项目的品牌视觉资源清单，而是高层组件消费视觉语义时共同依赖的底层结构语言，把“设计 token、主题映射、密度档位、状态色调、可读回退”收敛成前端可以稳定消费的最小视觉语法。这里的 `TOKEN` 指跨主题稳定存在的视觉语义键，这里的 `THEME` 指这些语义键在主题上下文中的映射、局部覆盖与回退关系，不应把两者混写为同一层具体取值表。

直观示例：按钮的视觉语义闭环

以一个常见的主按钮为例：

- 文本、背景、边框、间距与层级先对应稳定语义令牌。
- 同一组令牌可在浅色主题、深色主题与局部覆盖中映射为不同取值。
- 在紧凑、常规与宽松密度下，按钮尺寸与间距按稳定密度档变化。
- 在默认、悬停、按下、禁用与危险态下，色调映射关系需稳定。
- 在不同主题与密度组合下，对比、字号、焦点可见与视觉回退链都应满足可读约束。

选择语义令牌
  -> 映射到当前主题
  -> 按密度档调整尺寸
  -> 附着状态色调
  -> 校验可读性

这个例子说明：本模块不是规定某个组件最终长什么样，而是在规定高层组件至少要先形成“令牌、主题、密度、色调与可读性”共同闭合的视觉基础结构，才能稳定复用。

## 1. 能力声明（Capability Statement）

- `C1` 语义令牌组织能力：把排版、色彩、间距与层级语义键收敛为稳定视觉结构。
- `C2` 主题配方映射能力：把主题集、状态色调与变体配方收敛为稳定主题结构。
- `C3` 可读约束承接能力：把密度、对比与可读性回退链收敛为稳定视觉约束结构。

### 非职责声明（Non-Responsibility Statement）

- `N1` 非职责声明：不负责宿主挂载、页面流程与实例品牌文案。

## 2. 边界定义（Boundary / 参数）

- `TOKEN` 令牌边界：排版、色彩、间距、层级语义键与令牌映射关系需稳定。来源：`C1 + C3`。
- `THEME` 主题边界：主题集、状态映射、局部覆盖作用域与回退链需稳定。来源：`C1 + C2`。
- `DENSITY` 密度边界：紧凑、常规与宽松密度档位、密度切换条件与密度对布局影响范围需稳定。来源：`C1 + C3`。
- `TONE` 色调边界：主色调、辅助色调、状态色调关系与色调变体映射条件需稳定。来源：`C2 + C3`。
- `A11Y` 可读边界：对比约束、可读字号、焦点可见提示与视觉回退链需满足可访问要求；本边界只负责视觉可读约束，不替代承载与绑定层的结构可达定义。来源：`C2 + C3`。

## 3. 最小可行基（Minimum Viable Bases）

- `B1` 语义令牌结构基：由排版、色彩、间距、层级语义键与密度档构成的视觉语义结构。来源：`C1 + TOKEN + DENSITY`。
- `B2` 主题配方结构基：由主题集、状态色调、局部覆盖回退链与可读约束构成的主题配方结构。来源：`C2 + C3 + THEME + TONE + A11Y`。

## 4. 基组合原则（Base Combination Principles）

- `R1` 令牌先行
  - `R1.1` 参与基：`B1`。
  - `R1.2` 组合方式：先固定令牌键、令牌映射关系与密度档，再允许更高层组件消费这些视觉语义。
  - `R1.3` 输出能力：`C1`。
  - `R1.4` 边界绑定：`TOKEN + DENSITY`。
- `R2` 主题闭合
  - `R2.1` 参与基：`B1 + B2`。
  - `R2.2` 组合方式：在令牌语义上叠加主题配方、状态色调、局部覆盖作用域与可读回退链，形成稳定视觉基础层。
  - `R2.3` 输出能力：`C1 + C2 + C3`。
  - `R2.4` 边界绑定：`THEME + TONE + A11Y + TOKEN + DENSITY`。
- `R3` 禁止组合
  - `R3.1` 参与基：`B1 + B2`。
  - `R3.2` 组合方式：没有令牌语义就直接声明主题变体、色调关系不可解释、局部覆盖作用域失控或可读回退链缺位的方案均无效。
  - `R3.3` 失效结论：`N1`。
  - `R3.4` 边界绑定：`TOKEN + THEME + DENSITY + TONE + A11Y`。

## 5. 验证（Verification）

- `V1` 令牌完整性：视觉令牌结构必须满足 `TOKEN + DENSITY` 约束；排版、色彩、间距与层级语义键完整，且令牌映射关系与密度档位可判定。
- `V2` 主题闭合有效性：主题结构必须满足 `THEME + TONE` 约束；主题集、状态色调与变体配方可解释，且局部覆盖作用域与回退链完整。
- `V3` 可读一致性：可读约束结构必须满足 `DENSITY + A11Y` 约束；密度切换条件明确，对比与字号满足可访问要求，且焦点可见提示与视觉回退链可解释。
- `V4` 最小必要性：移除任一 `B*` 后，`V1 ~ V3` 至少一项失败。
- `V5` 结论表达：逐条输出 `R* -> C* + 边界绑定` 或 `R* -> N* + 边界绑定` 的通过或失败结论，并附对应边界证据。
