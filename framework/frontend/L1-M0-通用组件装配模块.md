# 前端通用组件装配模块:FrontendGenericComponentAssembly

@framework

本模块在前端通用框架内部承担“原子到组件装配”的角色。更底层的按钮、输入框、文本块、列表、树、标签、气泡等结构位于 `L0`；本模块只负责把这些原子与令牌、运行支撑装配为稳定通用组件面。

## 1. 能力声明（Capability Statement）

- `C1` 通用组件装配能力：把输入、承载、列表、层级、标签与反馈原子装配为稳定通用组件。
- `C2` 主题运行支撑能力：把组件装配面与令牌、主题、挂载和治理支撑统一到同一结构骨架中。
- `C3` 上游承接输出能力：为更高层前端标准与领域框架输出稳定可插接的通用组件面。
- `C4` 非能力项：不负责领域对象语义、页面场景流与后端契约执行。

## 2. 边界定义（Boundary / 参数）

- `COMPOSE` 装配边界：输入原子、承载原子、层级原子与反馈原子的装配职责需稳定。来源：`C1 + C2`。
- `SURFACE` 承载面边界：表单面、列表面、树面、卡片面与消息面需稳定输出。来源：`C1 + C3`。
- `TOKEN` 令牌边界：通用组件只能消费语义令牌，不得写死实例样式。来源：`C1 + C2`。
- `RUNTIME` 运行边界：挂载、缓存、回滚与治理支撑需覆盖组件装配面。来源：`C2 + C3`。
- `EVENT` 事件边界：点击、提交、选择、展开与上传等出口需一致。来源：`C1 + C2`。
- `A11Y` 可访问边界：装配后的通用组件必须具备稳定焦点、提示与阅读关系。来源：`C1 + C2`。

## 3. 最小可行基（Minimum Viable Bases）

- `B1` 输入承载装配基：L0.M0[R1,R2] + L0.M1[R1,R2]。来源：`C1 + COMPOSE + SURFACE + EVENT`。
- `B2` 列表反馈装配基：L0.M2[R1,R3] + L0.M3[R1,R3]。来源：`C1 + COMPOSE + SURFACE + A11Y`。
- `B3` 主题运行支撑装配基：L0.M0[R2,R3] + L0.M1[R2,R3] + L0.M4[R1,R3] + L0.M5[R1,R3]。来源：`C2 + C3 + TOKEN + RUNTIME + EVENT`。

## 4. 基组合原则（Base Combination Principles）

- `R1` 通用组件面收敛
  - `R1.1` 参与基：`B1 + B2`。
  - `R1.2` 组合方式：先把输入承载装配面与列表反馈装配面对齐，形成稳定通用组件面。
  - `R1.3` 输出能力：`C1`。
  - `R1.4` 边界绑定：`COMPOSE/SURFACE/EVENT/A11Y`。
- `R2` 主题运行统一
  - `R2.1` 参与基：`B1 + B3`。
  - `R2.2` 组合方式：把输入承载装配面绑定到令牌、主题、挂载与治理支撑，使通用组件具有稳定主题和运行承接。
  - `R2.3` 输出能力：`C1 + C2`。
  - `R2.4` 边界绑定：`TOKEN/RUNTIME/EVENT/SURFACE`。
- `R3` 上游插接面输出
  - `R3.1` 参与基：`B1 + B2 + B3`。
  - `R3.2` 组合方式：联合装配面、反馈面和主题运行支撑，输出可供更高层前端和领域框架直接承接的通用组件面。
  - `R3.3` 输出能力：`C1 + C2 + C3`。
  - `R3.4` 边界绑定：`COMPOSE/SURFACE/TOKEN/RUNTIME/EVENT/A11Y`。
- `R4` 禁止组合
  - `R4.1` 参与基：`B1 + B2 + B3`。
  - `R4.2` 组合方式：绕过基础原子直接固化领域语义、跳过令牌与运行支撑或破坏事件出口一致性的方案均无效。
  - `R4.3` 输出能力：`C4`。
  - `R4.4` 边界绑定：`COMPOSE/SURFACE/TOKEN/RUNTIME/EVENT/A11Y`。

## 5. 验证（Verification）

- `V1` 装配覆盖性：输入承载、列表层级和反馈表达三类装配面必须同时具备。
- `V2` 事件一致性：同类交互事件在不同通用组件中的输入输出契约必须一致。
- `V3` 支撑符合性：所有通用组件必须同时满足 `TOKEN` 与 `RUNTIME` 支撑约束。
- `V4` 可访问符合性：有效组合必须满足 `A11Y` 约束。
- `V5` 最小必要性：移除任一 `B*` 后，`V2~V4` 至少一项失败。
- `V6` 结论表达：逐条输出 `R* -> C* / 边界` 的通过或失败结论。
