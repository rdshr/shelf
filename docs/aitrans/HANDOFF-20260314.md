本轮工作已经把 `AiTrans` 从“框架与项目实例已定义”推进到“Windows 安装版可运行、可更新、主链可闭环、界面仍需继续打磨”的状态。

### 3.2 OCR 与翻译

OCR 当前实现：

- 主链：`tesseract`
- 升级入口保留：`windows_ocr_api`
- 正式发布路径采用 bundled OCR runtime

已完成的 OCR 第一轮优化：

- 避免默认 `zh/en/ja` 全量混跑
- 加入预处理变体：原图、2x 灰度、2x 阈值
- `--psm` 改成按截图形态自适应尝试

翻译当前实现：

- Provider 名：`openai_translation`
- 实际语义：OpenAI-compatible endpoint
- 默认模型：`qwen3-30b-a3b-instruct-2507`

支持来源：

- OpenAI 官方
- 局域网兼容端点

### 3.3 打包与发布

Windows 打包链已完成这些收口：

- `dist/` 清理脚本已加，避免多版本产物混杂

- `stage:tesseract` 会复制 `tesseract.exe + *.dll + tessdata`
- `release:check` 会校验 OCR runtime、release notes、版本一致性
- `release:manifest` 已支持按 channel 识别 `stable.yml`
- 安装器已改为 `per-user`，不再要求管理员运行

关键脚本：

- `apps/desktop_screenshot_translate/electron/scripts/clean-dist.js`
- `apps/desktop_screenshot_translate/electron/scripts/materialize-project.js`
- `apps/desktop_screenshot_translate/electron/scripts/stage-tesseract.js`
- `apps/desktop_screenshot_translate/electron/scripts/release-check.js`
- `apps/desktop_screenshot_translate/electron/scripts/release-manifest.js`
