# 置物架框架标准（L2）

## 0. 规范关系
本文件为置物架领域标准，必须遵循：
- `standards/L0/规范总纲与树形结构.md`
- `standards/L1/框架设计核心标准.md`
- `standards/L1/可追溯性标准.md`
- `standards/L1/可删减性标准.md`
本文件位于 `L2`，承接 `L1` 方法标准并向 `L3` 实现层传导。

## 1. 目标（Goal）
定义可拆卸组装的开放式立体堆叠装置，以提升单位占地面积下的存取效率。

## 2. 边界定义（Boundary）
- `N`：层数（`BoundaryDefinition.layers_n`）
- `P`：每层承重（`BoundaryDefinition.payload_p_per_layer`）
- `S`：每层空间（`BoundaryDefinition.space_s_per_layer`）
- `O`：开口尺寸（`BoundaryDefinition.opening_o`）
- `A`：占地面积（`BoundaryDefinition.footprint_a`）

## 3. 模块（最小可行基，Module）
- `M1` 杆（`Module.ROD`）：结构骨架与承力通道
- `M2` 连接接口（`Module.CONNECTOR`）：对接关系，传力规则与装配语义
- `M3` 隔板（`Module.PANEL`）：提供承载面与把载荷分给连接点

## 4. 组合原则（Combination Principles）
- `R1`：模块组合不得孤立，至少包含 2 个模块（len(module_combo) >= 2）
- `R2`：模块组合必须包含连接接口模块（Module.CONNECTOR in module_combo）
- `R3`：结构实例有效的必要条件之一是层数至少为 2（layers_n >= 2）
- `R4`：结构实例中所有杆仅允许沿 Z 轴放置，且杆与隔板相互垂直
- `R5`：结构实例中所有隔板两两平行，每块隔板四角必须通过连接接口与杆相连
- `R6`：有效组合集合必须仅包含同时满足 R1~R5 ，任一不满足即判定为无效，并可枚举有效组合集合
- `R7`：候选组合可包含不满足规则的样本用于验证与对比，最终输出结果只包含判定为有效的组合

## 5. 验证（Verification）
通过条件：
- 边界有效
- 组合属于有效组合集
- `target_efficiency > baseline_efficiency`

## 6. 领域映射（L2 -> L3）
- Goal -> `Goal`
- Boundary -> `BoundaryDefinition`, `Space3D`, `Opening2D`, `Footprint2D`
- Module -> `Module`, `MODULE_ROLE`
- Combination Principles -> `Rule`, `CombinationRules`
- Verification -> `VerificationInput`, `VerificationResult`, `verify()`
