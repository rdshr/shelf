# 置物架框架标准（L2）

## 0. 规范关系
本文件为置物架领域标准，必须遵循：
- `standards/L0/规范总纲与树形结构.md`
- `standards/L1/框架设计核心标准.md`
- `standards/L1/可追溯性标准.md`
- `standards/L1/可删减性标准.md`
本文件位于 `L2`，承接 `L1` 方法标准并向 `L3` 实现层传导。

## 1. 目标（Goal）
定义可拆卸组装的开放式立体堆叠装置，以提升单位占地面积下的存取效率。

## 1.1.适用范围（Scope）
- 单位面积下的物品存取效率场景
- 不考虑载重问题，即待存取物物品重量不超过装置承力能力
- 不考虑开口空间内的物品存取能力问题，即待存取物物品不超出装置开口空间
- 适用于“板面由杆系角点支撑”的系统；

## 2. 边界定义（Boundary Definition）
- `N`：层位点上限（`BoundaryDefinition.layers_n`），层索引为 `0..N-1`
- `A`：占地面积（`BoundaryDefinition.footprint_a`），按投影实际占用位点计算
- `S_panel_0`：0层板面面积
- `S_panel_above`：上层板面面积（层索引 `>=1`）
- `S_use`：取用面积总量，`S_use = A + S_panel_above`（0层板与地面互斥计入）
- `G_baseline`：基线取用增益（`BoundaryDefinition.baseline_gain`）

## 3. 模块（最小可行基，Module）
- `M1` 杆（`Module.ROD`）：结构承力通道
- `M2` 连接件（`Module.CONNECTOR`）：对接关系，传力规则与装配语义
- `M3` 板（`Module.PANEL`）：提供承载面与把载荷分给连接点

## 4. 组合原则（Combination Principles）
- `R1`：构件不可堆叠。
- `R2`：构件不得穿透。
- `R3`：有效组合至少包含 1 根杆和 1 块板。
- `R4`：连接仅允许通过连接件。
- `R5`：杆与板必须正交放置。
- `R6`：同一基准方向内承重杆应平行，连续延伸可共线。
- `R7`：板仅角点端接，杆仅端点端接。
- `R8`：有效组合必须为单一连通整体。
- `R9`：杆仅保留至其端接板的最高层；
- `R10`：所有层板的4个角点必须全部与杆端点端接。

## 5. 验证（Verification）
通过条件：
- 边界有效
- 组合属于有效组合集
- `support_mask(z=0) ⊆ support_mask(z>=1)`
- `efficiency_current = S_use / A`
- `gain_ratio = efficiency_current / G_baseline`
- `gain_ratio > 1`
