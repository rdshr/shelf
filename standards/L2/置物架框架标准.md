# 置物架框架标准（L2）

## 0. 规范关系
本文件为置物架领域标准，必须遵循：
- `standards/L0/规范总纲与树形结构.md`
- `standards/L1/框架设计核心标准.md`
- `standards/L1/可追溯性标准.md`
- `standards/L1/可删减性标准.md`
本文件位于 `L2`，承接 `L1` 方法标准并向 `L3` 实现层传导。
术语统一：
- `M1: 线` -> `Module.ROD`
- `M2: 面` -> `Module.PANEL`
- `M3: 接口` -> `Module.CONNECTOR`

## 1. 目标（Goal）
定义可拆卸组装的开放式立体堆叠装置，在给定占地面积条件下提升物品存取效率。
目标对象：
- `Goal.statement`
目标指标：
- `baseline_efficiency`（基线存取效率）
- `target_efficiency`（目标存取效率）
- 通过关系：`target_efficiency > baseline_efficiency`

## 2. 边界定义（Boundary）
- `N`：层数（`BoundaryDefinition.layers_n`）
- `P`：每层承重（`BoundaryDefinition.payload_p_per_layer`）
- `S`：每层空间（`BoundaryDefinition.space_s_per_layer`）
- `O`：开口尺寸（`BoundaryDefinition.opening_o`）
- `A`：占地面积（`BoundaryDefinition.footprint_a`）
硬约束：
- `layers_n > 0`
- `payload_p_per_layer > 0`
- `space_s_per_layer.width/depth/height > 0`
- `opening_o.width/height > 0`
- `footprint_a.width/depth > 0`
范围约束（工程化扩展）：
- `N_min <= layers_n <= N_max`
- `H_min <= space_s_per_layer.height <= H_max`
- `A_min <= (space_s_per_layer.width * space_s_per_layer.depth) <= A_max`
- `O_min <= (opening_o.width * opening_o.height) <= O_max`
- 连接关系需支持可逆拆装

## 3. 模块（最小可行基，Module）
- `M1` 杆（`Module.ROD`）：结构骨架与承力通道
- `M2` 连接接口（`Module.CONNECTOR`）：定义连接语义、对齐与载荷传递
- `M3` 隔板（`Module.PANEL`）：提供承载面并将载荷分配到连接点
最小属性建议：
- `Module.ROD`：`length`, `direction_vector`, `head_point`
- `Module.CONNECTOR`：`connector_type`, `supported_joint_types`
- `Module.PANEL`：`width/depth` 或 `polygon_vertices`
可选扩展属性：
- 材质、质量、抗压/抗弯/抗剪指标
- 防腐等级、制造公差

## 4. 组合原则（Combination Principles）
- `R1`：组合不得孤立，至少包含 2 个模块（对应：`len(combo) >= 2`）
- `R2`：每个可用组合必须包含连接接口模块（对应：`Module.CONNECTOR in combo`）
- 组合结果必须可判定为“有效/无效”，并可枚举有效组合集合（`CombinationRules.valid_subsets`）
工程强制规则：
- `E1`：至少 2 层平行 `Module.PANEL` 形成分层承载结构
- `E2`：每层 `Module.PANEL` 至少连接 2 个支撑单元（`Module.ROD` 或等效支撑件）
- `E3`：任意 `Module.ROD` 与 `Module.PANEL` 连接必须通过 `Module.CONNECTOR`
- `E4`：全结构必须单连通，不允许孤立子结构
- `E5`：层数、层高、单层面积必须落在边界范围内
- `E6`：禁止非接口定义的穿透、非法相交、悬空连接
推荐规则（可删减项）：
- `S1`：`Module.ROD` 与 `Module.PANEL` 优先垂直
- `S2`：每层优先 4 支撑点
- `S3`：连接器优先位于板角点或预定义安装位
- `S4`：多层轮廓优先对齐（同形或小偏移）
- `S5`：开口方向与比例按场景偏好配置

## 5. 验证（Verification）
通过条件：
- 边界有效
- 组合属于有效组合集
- `target_efficiency > baseline_efficiency`
验证输入输出：
- 输入：`VerificationInput(boundary, combo, valid_combinations, baseline_efficiency, target_efficiency)`
- 输出：`VerificationResult(boundary_valid, combination_valid, efficiency_improved, passed, reasons)`
工程试验项：
- `V-DETACH`：拆卸/复装验证
- `V-OPEN`：开放式通道验证（无需开门/抽屉/翻盖）
- `V-STATIC`：静载稳定性验证
- `V-DYNAMIC`：动载抗倾覆验证
- `V-EFF`：同占地面积下步骤数与耗时对比验证
失败归档建议：
- `BND_*`, `COMBO_*`, `EFF_*`, `MECH_*`, `DETACH_*`
